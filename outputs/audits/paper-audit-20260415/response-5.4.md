**Methodological recommendations**
- Replace exact-string grading with alias-aware scoring for structured labels.
- Add an asset validator that rejects any `wrong_scaffold` containing refusal language, gold-answer substrings, or residual `FINAL:`/“Correct answer” text.
- When a scorer changes, rerun the full analysis stack, not only the cell that triggered the audit.
- Add a stratified manual adjudication sample per `(model, condition)` cell, oversampling prefill cells and structured-label exact questions.
